## Project Description
A Django web application that generates mood-based music using AI logic and audio samples. Features user interface, categorized WAV samples, and dynamic music creation system.

## Project Structure
- **core**: Core application files
- **musicgen**: Music generation logic
- **db.sqlite3**: Database file
- **manage.py**: Django management script

## Technologies Used
- Django
- Python
- AI-based music generation
- Audio processing
